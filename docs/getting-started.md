# Getting started

SANG is built with Expo, so you can run it on your own phone without any complex
setup.

## What you need
- A computer with the project installed.
- A smartphone (iPhone or Android) with the free **Expo Go** app installed from the
  App Store or Google Play.
- Both devices on the same Wi-Fi network.

## Running the app
1. Start the project from the computer (your developer will run a single start
   command).
2. A QR code appears in the terminal.
3. On your phone, open **Expo Go** and scan that QR code.
4. SANG opens on your phone.

> The scanning feature needs a real phone, because phone simulators on a computer
> can't use a camera.

## Trying the full experience
To see the share-and-save flow the way it's meant to work, use **two phones**, each
running SANG:

1. Finish the welcome screen on both phones (enter name, number, email).
2. On the first phone, open **My Card** to show its QR code.
3. On the second phone, open **Scan** and point it at the first phone's code.
4. The first person is now saved in the second phone's **Contacts**.

That's the whole idea of SANG — two taps to share, one scan to save.
