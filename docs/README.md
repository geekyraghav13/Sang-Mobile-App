# SANG — Documentation

SANG is a mobile app that turns your contact details into a QR code other people
can scan to instantly save you. No business cards, no typing — just point a phone
at a code.

This folder explains what SANG is, how it works, and how to run it. It is written
for anyone — you don't need to be a developer to understand it.
